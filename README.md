# Galaxyhub BC
## 소개
Galaxyhub BC 프로젝트는 바 시티즌 코리아의 웹사이트로 국내외 바 시티즌 코리아 홍보를 위해 제작되었습니다. 
바 시티즌 행사 참가 신청, 입장권 발송, 행사 알림등 다양한 기능들이 지속적으로 추가될 예정입니다.

## 기능
- GitHub Repository 연동기능
